# React + TypeScript + Next

A mobile platform-adventure game where the player jumps from platform to platform, collects stars, and unlocks new worlds.

## Technologies

- Next.js 
- React  
- TypeScript 
- Tailwind CSS
- MongoDB

## Installation

```
npm install
```

## Usage

```
# Start development
npm run dev

# Exposes your local app to the internet
npm run devexp
```

## Requirements

- Node.js 20.0.0 or higher
- npm

## Project Structure

```
/app            # Backend
/components     # React components
/prisma         # Prisma schema DB
```
